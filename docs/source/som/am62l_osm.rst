********************
General Information
********************

   The **byteENGINE AM62L OSM-S** is a high performance industrial oriented computing module. It allows you a short time-to-market, reducing development costs and substantial design risks. 
   
   The **System on Module (SOM)** uses the AM62L devices which are based on the high-performance dual ARM® Cortex®-A53 64-bit core operating at up to 1.25GHz. The AM62L devices are built for embedded Linux applications with extensive security options and optimized power architecture. The AM62L also includes an extensive set of peripherals including modern display interfaces, making the AM62L a well-suited general purpose device for a broad range of industrial applications.
   
   **The OSM-S standard** is used by the  byteENGINE AM62L OSM-S to allow soldering to the carrier board with 332 contacts following the Open Standard Module design to allow for cost efficient and versatile mouting options.
