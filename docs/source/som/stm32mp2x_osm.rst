********************
General Information
********************

   The **byteENGINE STM32MP2 OSM-S** is a high performance industrial oriented computing module. It allows you a short time-to-market, reducing development costs and substantial design risks. 
   
   The **System on Module (SOM)** uses the STM32MP2 devices which are based on the high-performance dual ARM® Cortex®-A35 64-bit core operating at up to 1.5GHz.

   The **byteENGINE STM32MP2 OSM-S** devices also embed a Cortex®-M33 running at 400 Mhz and offer an embedded Verisilicon 3D graphic processing unit running up to 900 MHz, with performances up to 150 MTriangle/s, 900 Mpixel/s. Artificial Intelligence is supported with a VeriSilicon NPU running up to 900Mhz, supporting TensorFlowLite - ONNX and Linux NN.

   **The OSM-S standard** is used by the byteENGINE STM32MP2 OSM-S to allow soldering to the carrier board with 332 contacts following the Open Standard Module design to allow for cost efficient and versatile mouting options.
